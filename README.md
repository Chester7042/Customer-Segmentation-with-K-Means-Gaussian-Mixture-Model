# Customer Segmentation with K Means Gaussian Mixture Model
Proyek ini berfokus pada customer segmentation menggunakan pendekatan unsupervised learning untuk mengelompokkan pelanggan berdasarkan pola karakteristik dan perilaku mereka. Data terlebih dahulu melalui tahap exploratory data analysis dan preprocessing, termasuk feature scaling dengan StandardScaler agar setiap variabel berada pada skala yang sebanding. Untuk meningkatkan interpretabilitas dan efisiensi komputasi, dilakukan reduksi dimensi menggunakan Principal Component Analysis (PCA) sebelum proses clustering.

Dua algoritma clustering utama diterapkan, yaitu K-Means dan Gaussian Mixture Model (GMM). Jumlah cluster optimal ditentukan menggunakan Elbow Method dan Silhouette Score pada K-Means, kemudian hasilnya dibandingkan dengan GMM untuk melihat perbedaan struktur klaster berbasis distribusi probabilistik. Visualisasi hasil clustering dilakukan dalam ruang PCA dua dan tiga dimensi untuk memudahkan analisis pola antar cluster. Proyek ini menghasilkan segmentasi pelanggan yang jelas dan dapat digunakan sebagai dasar pengambilan keputusan bisnis, seperti strategi pemasaran yang lebih terarah dan personalisasi layanan.
